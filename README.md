# jonclarkdevelopments.com

Jon Clark Developments — property listing site. Static, served via GitHub Pages.

- `index.html` — the listing (built from the North State St page)
- `edit/` — reviewer annotation tool (noindex)
- `img/`, `video/`, `gallery.json` — media + gallery manifest
- `assets/vendor/` — self-hosted Bootstrap CSS, Swiper, GLightbox, AOS, Bootstrap Icons
- `CNAME` — custom domain

Rebuild with `build_jon_site.py` (in the Jon working folder).
